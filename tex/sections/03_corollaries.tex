\section{Corollaries}
\label{sec:corollaries}

\noindent
This section collects short, compute–first consequences of the Stokes/continuity identity
\(d(*j)=\Sigma\) together with the snapshot static closure (when invoked), organized as a ladder.
The core rungs (C0–C7) are where most readers should start; C8–C12 are optional extensions that
build on the same spine.

\begin{itemize}\itemsep2pt
  \item \textbf{Core (C0–C7):} bit–gas on screens; static inverse–area law; continuity \& Clausius;
  external drives with delay/production; Info–EM waves \& export; time–information bound;
  noise floor; geometry as a consistency tool.
  \item \textbf{Extensions (C8–C12):} dimensional flows \& confinement (1D/2D);
  Thermo $\to$ Newtonian rate laws; least–action equivalence (derived);
  entropic–gravity corrections (attenuation/capacity ceilings); quantum path–integral template.
\end{itemize}
